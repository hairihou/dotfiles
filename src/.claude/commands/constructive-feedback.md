# Constructive Feedback (SBI Framework)

Transform the input into constructive feedback using the SBI (Situation-Behavior-Impact) framework.

## Input

$ARGUMENTS

## Transform

Convert the input into the output format:

- Add constructive suggestions for improvement
- Make vague points specific
- Replace emotional expressions with fact-based statements

## SBI Framework

| Element       | Description                    | Example                                               |
| ------------- | ------------------------------ | ----------------------------------------------------- |
| **Situation** | Specific context (when, where) | "During last Monday's team meeting"                   |
| **Behavior**  | Observable action (facts only) | "The report was submitted 3 days late"                |
| **Impact**    | Effect of the behavior         | "This delayed the client proposal and affected trust" |

## Output Format

```
【Situation】
【Behavior】
【Impact】

---

【Suggestion】
【Expected Outcome】
```

## Guidelines

- **Be specific**: Avoid vague terms ("always", "never"); use concrete examples
- **Focus on behavior, not personality**: Never attack character
- **Future-oriented**: Focus on improvement, not past blame
- **Respect**: Honor the recipient's perspective
- **Separate facts from interpretation**: Use "X happened" not "I think X"
