# dotfiles

## Install Packages from Brewfile

```zsh
   brew bundle --file=./public/Brewfile
   # work
   brew bundle --file=./public/Brewfile.work.rb
```

## Sync All Settings and Configurations

```zsh
  make sync-all
```
